# Глава 6: Образовательная среда

## Ключевая идея
Образовательная среда (learning environment) — всё, с чем взаимодействует учащийся: физическое, цифровое, социальное окружение + ценности. Среда кодирует ценности и нормы (часто самопроизвольно) и определяет психологический климат — важнейший фактор результатов.

## Введённые фреймворки

- **Три компонента среды (В. А. Ясвин)**: пространственно-предметный (физический + цифровой), социальный, организационно-технологический (= методики, в этой главе опускается).

- **Среда в 4 идеологиях** — ценности задают облик среды:
  - **Scholar Academic** — библиотека как центр, дискуссии-гостиные, открытость знания.
  - **Social Efficiency** — сжатые сроки, прикладной контент, связь с индустрией, портфолио.
  - **Learner Centered** — персонализированное гибкое пространство, «слоты» учеников, учитель корректирует программу.
  - **Social Reconstruction** — дискуссионные клубы, публикация артефактов для общества.

- **Психологический климат** — субъективно воспринимаемая производная среды («дух школы» — Толстой). Здоровый климат (CSCI, Колумбийский ун-т) — факторы: **безопасность** (ключевой!), поддерживающее обучение, уважительные отношения, самоидентификация с ценностями, опрятность среды, инклюзивность, проявленное лидерство, профессиональные взаимоотношения команды.
  - Безопасность по аудиториям: физическая (симуляции — медики, пилоты), эмоциональная (подростки), прозрачность правил (взрослые, андрагогика).

- **Цифровая среда** — 3 параметра (ползунки):
  - **Автономность** (когортное ↔ самостоятельное/self-paced; синхронное ↔ асинхронное). Высокая автономность → среда должна быть **навигирующей** (ограничивающей), **поддерживающей** (self-directed + малый рост сложности), **прозрачной**.
  - **Количество студентов в моменте** — большие группы → потенциал взаимообучения (peer-to-peer); малые + высокая автономность → больше автоматизированных методов оценки.
  - **Длительность** — связана с вовлечённостью; длинные → понижать автономность, усиливать соцвзаимодействие (3 типа взаимодействия Дж. Мура: ученик–ученик, ученик–учитель, ученик–материал).
  - Среды: **физическая**, **цифровая (онлайн)**, **смешанная (blended)** — последовательное чередование, **гибридная (hybrid)** — часть группы онлайн, часть офлайн одновременно.
  - + Требования: соответствие уровню цифровой грамотности и матбазы; цифровая среда = потенциал для аналитики и персонализации.

- **Физическая среда** — пространственно-предметный компонент. Параметры значимости (Н. Иванова): гигиенический, эстетический, технический, персонализированность, безопасность, символический фактор. + **Пространство за пределами школы** — взаимосвязи с внешней средой (библиотеки, локальные сообщества); принципы: доступность/известность, прописанные правила, связь с целями.

- **Социальная среда и роли** — решает 3 задачи: помочь найти правильный контент (коллективные карты знаний), правильных людей (визуализация людей + нормы взаимодействия + раппорт/доверие), вдохновение/мотивация (признание, статусы).
  - **Раппорт** — доверительные отношения (необычная внимательность, установление связи, единое инфополе, уважение, общее с преподавателем).
  - Различай: социальный **компонент среды** (поддержка взаимодействия) vs социальное **обучение** (Бандура: копирование ролевых моделей; «скрытый учебный план / hidden curriculum»).

- **4 типа ролей в обучении**: **экспертные** (содержание: преподаватели, лекторы, индустрия), **поддерживающие** (тьюторы, фасилитаторы), **развивающие** (менторы, наставники), **координационные** (кураторы, администраторы, модераторы). См. Приложение 8.

## Ключевые понятия
- **Школьная атмосфера / культура школы / негласное образование (tacit education).**
- **Социализация / интериоризация** ценностей микросоциума.
- **Теория разбитых окон** — наблюдаемое нарушение провоцирует новые.
- **Раппорт, retention** (возвращаемость), **digital natives**.

## Анти-паттерны
- **Игнорировать ценностно-культурный компонент** — он складывается самопроизвольно, хотим мы того или нет.
- **Резкий рост сложности при высокой автономности** — некому помочь, студент выпадает.
- **Один человек выполняет все роли** — устоявшаяся практика, но снижает эффективность/качество.

## Ключевые выводы
1. Намеренно закладывай ценностно-культурный компонент — иначе он сложится случайно.
2. Климат строится на безопасности; соотноси среду с факторами здорового климата (CSCI).
3. Цифровую среду проектируй по 3 ползункам (автономность, размер, длительность).
4. Социальный компонент = контент + люди + мотивация; различай его и социальное обучение.
5. Разделяй роли по 4 типам, не сваливай всё на одного.
